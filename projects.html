Project 4 - Google Maps API
View on Github

This is a Google Maps viewer with added panorama of one of my favorite locations, Nasa Space Station

Definitely a work-in-progress Google Maps API code, as in the future, I would love to add more features, such as direction to the multiple areas of Houston's Nasa Space Station, distance between them, and such. So far, both map and panomara will sync together asynchronous, meaning the panorama view will be updated wherever you place the marker on the Google Maps portion. To view this, you will have to add your personal Google Maps API key on the source code.
